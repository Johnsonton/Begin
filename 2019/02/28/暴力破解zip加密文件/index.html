暴力破解zip加密文件

问题描述

小明是90后，用他的生日8位数字作为密码压缩了这个zip文件。正确做法：写脚本/用字典生成工具生成字典，用工具（ZipPasswordTool、Advanced Zip Password Recovery等）/写脚本暴力破解即可。

解题思路：

小明的密码是由8位数字的生日密码构成，前三位是199
根据提示，用暴力破解工具即可破解生日密码

所需工具：

Ziperello软件或其他暴力破解软件

解题过程：

已知：

1） 密码为数字密码；
2） 密码位数为八位；
3） 密码前三位为199；

操作：

1） 使用软件打开压缩文件

2） 选择暴力破解

3） 设定字符集为固定字符集，字符类型为数字，最小密码和最大密码长度均设为8，起始密码199

4） 开始破解，得到密码

5）将所得密码输入该文件，进行解压缩，得到flag.txt,完成。
